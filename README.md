# ITC Website – Web Convener Assignment

This project is a multi-page responsive website created as part of the **ITC Web Convener Application Assignment**.

The goal of this project was to design and build a simple but interactive website that demonstrates my understanding of **HTML, CSS. 

---

- Inspiration

The basic idea and structure of the website are **partially inspired by the ITC website**, especially elements like the **navigation bar and general layout**.  
However, the implementation, styling, and structure were developed and modified by me while learning and experimenting with web development concepts.

---

I tried to make good use of various **CSS properties** to improve the design and appearance of the website.

I have tried to make sure that website is interactive .

# Design Elements & Resources

Some elements used in the website are sourced from the internet for learning and experimentation purposes:

- The **background image** used in the website was downloaded from the internet.
- A **3D robot model** used on the website was taken from Spline, along with its required CSS styling is taken from Spline.

---

## Challenges Faced

While building this website, I faced several challenges during development:

- **Responsiveness Issues:** Initially, the website was not properly responsive on different screen sizes. I had to adjust layout properties, spacing, and element positioning to make the pages look better on different devices.

- **Overuse of CSS Properties:** During development, I experimented with many CSS properties, and in some places I ended up using unnecessary or redundant styling. Later, I tried to clean the code and understand which properties were actually needed.

- **Integrating External Elements:** Adding elements like the 3D robot model and external fonts required additional styling adjustments to ensure they fit well with the overall design.

- **UI Consistency:** Maintaining consistent spacing, fonts, and colors across different sections of the website required multiple iterations.

These challenges helped me better understand **how CSS works, how layouts behave on different screens, and how to structure cleaner code.**
